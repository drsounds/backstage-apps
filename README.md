# Backstage Apps

This workspace contains a collection of plugins that adds support for HTML5 stitch apps into Backstage, and the possibility to create them through vibe coding.

## Plugins

The following plugins are included in this workspace:

- **[chord](./plugins/chord)**: The "Vibe Coding" plugin. It allows users to create new apps using AI-powered prompts and previews.
- **[apps](./plugins/apps)**: The "Stitch Apps" plugin. It handles the integration and display of HTML5 apps within Backstage and provides the frontend interface for managing (listing, creating, updating, deleting) the apps.

## Getting Started

To start the app, run:

```sh
yarn install
yarn start
```
